# Math对象

ECMAScript 还为保存数学公式和信息提供了一个公共位置，即Math 对象。

*Math 对象的属性*

| 属 性          | 说 明                |
| ------------ | ------------------ |
| Math.E       | 自然对数的底数，即常量e的值     |
| Math.LN10    | 10的自然对数            |
| Math.LN2     | 2的自然对数             |
| Math.LOG2E   | 以2为底e的对数           |
| Math.LOG10E  | 以10为底e的对数          |
| Math.PI      | π的值                |
| Math.SQRT1_2 | 1/2的平方根（即2的平方根的倒数） |
| Math.SQRT2   | 2的平方根              |
